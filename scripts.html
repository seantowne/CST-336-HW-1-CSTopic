Setting Permisions on a file

There are a lot of great tutorials for how to change file permisions. Here is one . For our purposes we only need chmod. Lets make a file called myScript and make it excecutable.. 

                        
                            $ touch myScript 
                            $ chmod +x myScrip

Putting Commands Into a File

There are much better ways of editing a script file, namely, text editors, but for the purposes of this tutorial we will just use echo. 

                        
                            $ echo "#! /bin/bash" >> myScript 
                            $ echo "echo Hello, World" >> myScript
                            $ cat myScript 
                            #! /bin/bash
                            echo Hello, World

Run The Script!

We just made a script! Lets run it using the './scriptName' syntax.. 

                        
                            $ ./myScript 
                            Hello, World

Our script simply prints out a friendly message, because, thats what we told it to do!
